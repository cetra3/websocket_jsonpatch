# Using Rust to Synchronize state between clients

This is companion code that follows from this article:
https://www.formlogic.com/blog/using-rust-to-synchronize-state-between-clients

[View demo](https://raw.githubusercontent.com/cetra3/websocket_jsonpatch/master/example.mp4)


## Developing

You will need:

* Rust
* Node/Yarn


In one terminal window, run up the backend:

```
cargo run
```

In another window, start the frontend:

```
cd front && yarn install && yarn dev
```
